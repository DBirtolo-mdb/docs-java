Overview
--------

In this guide, you can learn how to use aggregation expression operators with the 
{+driver-long+}. These operators allow you to perform aggregation operations with
discoverable, typesafe Java methods rather than BSON documents.
Because these operators also implement the fluent interface pattern, you can chain 
aggregation operations together in a simple and intuitive way without using additional 
variables.
